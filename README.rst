Sherlock C++
============

C++ version of `Sherlock <http://github.com/vmlaker/sherlock>`_.

Requirements
------------

You're gonna need `SCons <http://www.scons.org>`_:
::
   
   yum install scons


Usage
-----

Build the codes by running SCons:
::
   
   scons

Clean up after the build:
::

   scons -c
